# Git Worktree Manager 自动更新功能技术方案

## 1. 功能概述

为 Git Worktree Manager 添加自动更新功能，用户无需手动下载新版本即可完成升级。

### 功能点

- 启动时自动检查更新
- 手动检查更新（设置面板）
- 显示更新进度
- 支持后台下载 + 一键安装
- 跨平台支持（macOS/Windows/Linux）

## 2. 技术选型

使用 **Tauri 官方 Updater 插件** (`tauri-plugin-updater`)

### 为什么选择官方插件

| 方案 | 优点 | 缺点 |
|------|------|------|
| tauri-plugin-updater | 官方维护、跨平台、签名验证 | 需要配置签名密钥 |
| 自定义实现 | 完全可控 | 开发成本高、需处理各平台差异 |

## 3. 实现方案

### 3.1 后端改动

#### 3.1.1 添加依赖

```toml
# src-tauri/Cargo.toml
[dependencies]
tauri-plugin-updater = "2"
```

#### 3.1.2 注册插件

```rust
// src-tauri/src/lib.rs
fn run() {
    tauri::Builder::default()
        .plugin(tauri_plugin_updater::Builder::new().build())
        // ... 其他插件
        .run(tauri::generate_context!())
        .expect("error while running tauri application");
}
```

### 3.2 配置改动

#### 3.2.1 tauri.conf.json 添加 updater 配置

```json
{
  "plugins": {
    "updater": {
      "pubkey": "生成的公钥",
      "endpoints": [
        "https://github.com/zhuSilence/git-worktree-manager/releases/latest/download/{{target}}-{{arch}}.json"
      ]
    }
  }
}
```

### 3.3 CI/CD 改动

#### 3.3.1 生成签名密钥对（一次性）

```bash
# 生成私钥（保存到 GitHub Secrets）
tauri signer generate -w ~/.tauri/git-worktree-manager.key

# 公钥写入 tauri.conf.json
```

#### 3.3.2 修改 release.yml

每个平台构建时生成：
- 安装包
- 签名文件（.sig）
- 更新清单（.json）

示例更新清单格式：
```json
{
  "version": "0.0.6",
  "notes": "Bug fixes and improvements",
  "pub_date": "2026-03-20T00:00:00Z",
  "platforms": {
    "darwin-aarch64": {
      "signature": "dW50cnVzdGVkIGNvbW1lbnQ6IHNpZ25hdHVyZSBmcm9tIHRhdXJp...",
      "url": "https://github.com/zhuSilence/git-worktree-manager/releases/download/v0.0.6/Git.Worktree.Manager_0.0.6_aarch64.dmg"
    }
  }
}
```

### 3.4 前端改动

#### 3.4.1 新增组件

```
src/components/UpdateManager/
├── index.tsx              # 更新管理器入口
├── UpdateDialog.tsx       # 更新弹窗
├── UpdateProgress.tsx     # 下载进度
└── hooks/
    └── useUpdater.ts      # 更新逻辑 hook
```

#### 3.4.2 更新流程

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│   检查更新       │────▶│   有新版本?      │────▶│   显示更新弹窗   │
│   (启动/手动)    │     │   ↓ 无          │     │                 │
└─────────────────┘     └─────────────────┘     └─────────────────┘
                                                        │
                        ┌─────────────────┐              │ 用户确认
                        │   下载更新包     │◀─────────────┘
                        │   (显示进度)     │
                        └─────────────────┘
                                │
                        ┌─────────────────┐
                        │   验证签名       │
                        │   安装更新       │
                        └─────────────────┘
                                │
                        ┌─────────────────┐
                        │   重启应用       │
                        └─────────────────┘
```

#### 3.4.3 设置面板集成

在设置面板添加：
- 当前版本显示
- 检查更新按钮
- 自动检查开关（可选）

## 5. GitHub Secrets 配置

需要在 GitHub 仓库中添加：

| Secret | 说明 |
|--------|------|
| `TAURI_SIGNING_PRIVATE_KEY` | Tauri 签名私钥 |
| `TAURI_SIGNING_PRIVATE_KEY_PASSWORD` | 私钥密码（如有） |

## 7. 注意事项

### macOS 安全提示

自动更新后的首次启动可能仍会触发 Gatekeeper 提示，建议：
- 申请 Apple 开发者证书进行签名
- 或引导用户使用 Homebrew 安装（已支持）

### Windows SmartScreen

未签名的 Windows 应用会触发 SmartScreen 警告：
- 可购买代码签名证书
- 或使用自签名证书（仍有警告）
