## 一、技术选型

### 1.1 技术栈总览

```
┌─────────────────────────────────────────────────────────────┐
│                        前端 (UI Layer)                       │
│  React 18 + TypeScript + TailwindCSS + shadcn/ui           │
├─────────────────────────────────────────────────────────────┤
│                      桌面应用框架                             │
│                     Tauri 2.0 (Rust)                        │
├─────────────────────────────────────────────────────────────┤
│                       后端 (Core Layer)                      │
│            Rust + git2 (libgit2 bindings)                   │
├─────────────────────────────────────────────────────────────┤
│                       系统 API                               │
│           文件系统 / 进程管理 / Shell 集成                    │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 技术选型理由

| 技术 | 选型理由 |
|------|----------|
| **Tauri 2.0** | 比 Electron 小 10x，内存占用低，安全性好，支持跨平台 |
| **React 18** | 生态成熟，组件化开发，支持并发渲染 |
| **TypeScript** | 类型安全，IDE 支持好，大型项目必备 |
| **TailwindCSS** | 原子化 CSS，开发效率高，包体积小 |
| **shadcn/ui** | 高质量组件库，可定制性强 |
| **Rust (后端)** | 性能极高，内存安全，与 Tauri 无缝集成 |
| **git2** | Rust 的 libgit2 绑定，直接操作 Git 仓库 |

### 1.3 备选方案对比

| 方案 | 优点 | 缺点 | 结论 |
|------|------|------|------|
| Electron | 生态成熟 | 体积大(150MB+)，内存高 | ❌ 不选 |
| Tauri | 轻量(<10MB)，安全 | Rust 学习曲线 | ✅ 选择 |
| Flutter | 跨平台 | 桌面体验一般 | ❌ 不选 |

```rust
// src-tauri/src/services/git_service.rs

use git2::Repository;
use crate::models::worktree::{Worktree, WorktreeStatus, LastCommit};
use std::path::Path;

pub fn list_worktrees(repo_path: &str) -> Result<Vec<Worktree>, anyhow::Error> {
    let repo = Repository::open(repo_path)?;
    let worktrees = repo.worktrees()?;
    
    let mut result = Vec::new();
    
    // 主 worktree
    let main_worktree = get_main_worktree_info(&repo)?;
    result.push(main_worktree);
    
    // 链接 worktrees
    for name in &worktrees {
        if let Some(wt) = get_worktree_info(&repo, name)? {
            result.push(wt);
        }
    }
    
    Ok(result)
}

fn get_main_worktree_info(repo: &Repository) -> Result<Worktree, anyhow::Error> {
    let path = repo.path().parent()
        .ok_or_else(|| anyhow::anyhow!("Invalid repository path"))?
        .to_path_buf();
    
    let head = repo.head()?;
    let commit = head.peel_to_commit()?;
    
    let branch = head.shorthand().map(String::from);
    let status = get_worktree_status(repo)?;
    let last_commit = get_last_commit_info(&commit)?;
    
    Ok(Worktree {
        id: commit.id().to_string(),
        path: path.to_string_lossy().to_string(),
        branch,
        head: commit.id().to_string(),
        is_main: true,
        status,
        last_commit,
        locked: false,
        locked_reason: None,
        prunable: false,
        prunable_reason: None,
    })
}

fn get_worktree_status(repo: &Repository) -> Result<WorktreeStatus, anyhow::Error> {
    let statuses = repo.statuses(None)?;
    
    let has_conflicts = statuses.iter().any(|s| {
        s.status().contains(git2::Status::CONFLICTED)
    });
    
    if has_conflicts {
        return Ok(WorktreeStatus::Conflict);
    }
    
    let has_changes = statuses.iter().any(|s| {
        !s.status().is_empty() && !s.status().contains(git2::Status::IGNORED)
    });
    
    if has_changes {
        return Ok(WorktreeStatus::Dirty);
    }
    
    // 检查 detached HEAD
    if repo.head_detached()? {
        return Ok(WorktreeStatus::Detached);
    }
    
    Ok(WorktreeStatus::Clean)
}

fn get_last_commit_info(commit: &git2::Commit) -> Result<LastCommit, anyhow::Error> {
    let time = commit.time();
    let now = std::time::SystemTime::now()
        .duration_since(std::time::UNIX_EPOCH)?
        .as_seconds();
    
    let relative_time = format_relative_time(now as i64, time.seconds());
    
    Ok(LastCommit {
        hash: commit.id().to_string()[..7].to_string(),
        message: commit.summary()
            .unwrap_or("No message")
            .to_string(),
        author: commit.author().name()
            .unwrap_or("Unknown")
            .to_string(),
        relative_time,
    })
}

fn format_relative_time(now: i64, commit_time: i64) -> String {
    let diff = now - commit_time;
    
    if diff < 60 {
        "just now".to_string()
    } else if diff < 3600 {
        format!("{} minutes ago", diff / 60)
    } else if diff < 86400 {
        format!("{} hours ago", diff / 3600)
    } else if diff < 604800 {
        format!("{} days ago", diff / 86400)
    } else {
        format!("{} weeks ago", diff / 604800)
    }
}
```

```rust
// src-tauri/src/services/git_service.rs

pub fn create_worktree(
    repo_path: &str,
    branch: &str,
    target_path: &str,
    create_branch: bool,
    base_branch: Option<&str>,
) -> Result<Worktree, anyhow::Error> {
    let repo = Repository::open(repo_path)?;
    
    // 检查路径是否存在
    let path = Path::new(target_path);
    if path.exists() {
        return Err(anyhow::anyhow!("Path already exists: {}", target_path));
    }
    
    // 创建 worktree
    if create_branch {
        // 从 base_branch 创建新分支
        let base = base_branch.unwrap_or("HEAD");
        let base_commit = if base == "HEAD" {
            repo.head()?.peel_to_commit()?
        } else {
            repo.find_branch(base, git2::BranchType::Local)?
                .into_reference()
                .peel_to_commit()?
        };
        
        repo.worktree(
            branch,
            path,
            Some(git2::WorktreeAddOptions::new().reference(&repo.reference(
                &format!("refs/heads/{}", branch),
                base_commit.id(),
                true,
                "Creating branch for worktree",
            )?)),
        )?;
    } else {
        // 使用现有分支
        repo.worktree(branch, path, None)?;
    }
    
    // 返回新创建的 worktree 信息
    // ... (获取 worktree 信息逻辑)
}
```

```rust
// src-tauri/src/services/git_service.rs

pub fn delete_worktree(
    repo_path: &str,
    worktree_path: &str,
    force: bool,
) -> Result<(), anyhow::Error> {
    let repo = Repository::open(repo_path)?;
    
    // 检查是否有未提交的更改
    if !force {
        let wt_repo = Repository::open(worktree_path)?;
        let status = get_worktree_status(&wt_repo)?;
        
        if status != WorktreeStatus::Clean {
            return Err(anyhow::anyhow!(
                "Worktree has uncommitted changes. Use force=true to delete anyway."
            ));
        }
    }
    
    // 删除 worktree
    repo.worktree_prune(Some(git2::WorktreePruneOptions::new().valid(true)))?;
    
    // 删除目录
    std::fs::remove_dir_all(worktree_path)?;
    
    Ok(())
}
```

```rust
use std::process::Command;

pub fn open_in_ide(path: &str, ide: &str) -> Result<(), anyhow::Error> {
    let (program, args) = match ide {
        "vscode" => ("code", vec![path]),
        "vscode-insiders" => ("code-insiders", vec![path]),
        "cursor" => ("cursor", vec![path]),
        "webstorm" => ("webstorm", vec![path]),
        "intellij" => ("idea", vec![path]),
        _ => return Err(anyhow::anyhow!("Unsupported IDE: {}", ide)),
    };
    
    Command::new(program)
        .args(&args)
        .spawn()?;
    
    Ok(())
}

pub fn open_in_terminal(path: &str, terminal: &str) -> Result<(), anyhow::Error> {
    #[cfg(target_os = "macos")]
    {
        match terminal {
            "terminal" => {
                Command::new("open")
                    .args(["-a", "Terminal", path])
                    .spawn()?;
            }
            "iterm2" => {
                Command::new("open")
                    .args(["-a", "iTerm", path])
                    .spawn()?;
            }
            "warp" => {
                Command::new("open")
                    .args(["-a", "Warp", path])
                    .spawn()?;
            }
            _ => return Err(anyhow::anyhow!("Unsupported terminal: {}", terminal)),
        }
    }
    
    #[cfg(target_os = "windows")]
    {
        match terminal {
            "cmd" => {
                Command::new("cmd")
                    .args(["/C", "start", "cmd", "/K", &format!("cd {}", path)])
                    .spawn()?;
            }
            "powershell" => {
                Command::new("powershell")
                    .args(["-Command", &format!("Start-Process powershell -ArgumentList '-NoExit', '-Command', \"cd '{}'\"", path)])
                    .spawn()?;
            }
            "wt" => {
                Command::new("wt")
                    .args(["-d", path])
                    .spawn()?;
            }
            _ => return Err(anyhow::anyhow!("Unsupported terminal: {}", terminal)),
        }
    }
    
    #[cfg(target_os = "linux")]
    {
        match terminal {
            "gnome-terminal" => {
                Command::new("gnome-terminal")
                    .args(["--working-directory", path])
                    .spawn()?;
            }
            "alacritty" => {
                Command::new("alacritty")
                    .args(["--working-directory", path])
                    .spawn()?;
            }
            _ => return Err(anyhow::anyhow!("Unsupported terminal: {}", terminal)),
        }
    }
    
    Ok(())
}
```

```typescript
// stores/worktreeStore.ts

import { create } from 'zustand';
import type { Worktree, Repository } from '../types';
import * as gitService from '../services/git';

interface WorktreeState {
  // 状态
  repositories: Repository[];
  currentRepo: Repository | null;
  worktrees: Worktree[];
  isLoading: boolean;
  error: string | null;
  
  // 操作
  loadRepository: (path: string) => Promise<void>;
  refreshWorktrees: () => Promise<void>;
  createWorktree: (branch: string, path: string, createBranch?: boolean) => Promise<void>;
  deleteWorktree: (worktreePath: string, force?: boolean) => Promise<void>;
  setCurrentRepo: (repo: Repository | null) => void;
}

export const useWorktreeStore = create<WorktreeState>((set, get) => ({
  // 初始状态
  repositories: [],
  currentRepo: null,
  worktrees: [],
  isLoading: false,
  error: null,
  
  // 加载仓库
  loadRepository: async (path: string) => {
    set({ isLoading: true, error: null });
    
    try {
      const worktrees = await gitService.listWorktrees(path);
      const branches = await gitService.listBranches(path);
      
      const repo: Repository = {
        id: path,
        name: path.split('/').pop() || path,
        gitDir: `${path}/.git`,
        mainWorktreePath: path,
        worktrees,
        branches,
        remotes: [],
      };
      
      set({ 
        currentRepo: repo, 
        worktrees,
        isLoading: false 
      });
    } catch (error) {
      set({ 
        error: (error as Error).message, 
        isLoading: false 
      });
    }
  },
  
  // 刷新 worktrees
  refreshWorktrees: async () => {
    const { currentRepo } = get();
    if (!currentRepo) return;
    
    set({ isLoading: true });
    
    try {
      const worktrees = await gitService.listWorktrees(currentRepo.mainWorktreePath);
      set({ worktrees, isLoading: false });
    } catch (error) {
      set({ 
        error: (error as Error).message, 
        isLoading: false 
      });
    }
  },
  
  // 创建 worktree
  createWorktree: async (branch, path, createBranch = false) => {
    const { currentRepo } = get();
    if (!currentRepo) return;
    
    set({ isLoading: true });
    
    try {
      await gitService.createWorktree({
        repoPath: currentRepo.mainWorktreePath,
        branch,
        path,
        createBranch,
      });
      
      // 刷新列表
      await get().refreshWorktrees();
    } catch (error) {
      set({ 
        error: (error as Error).message, 
        isLoading: false 
      });
      throw error;
    }
  },
  
  // 删除 worktree
  deleteWorktree: async (worktreePath, force = false) => {
    const { currentRepo } = get();
    if (!currentRepo) return;
    
    set({ isLoading: true });
    
    try {
      await gitService.deleteWorktree({
        repoPath: currentRepo.mainWorktreePath,
        worktreePath,
        force,
      });
      
      // 刷新列表
      await get().refreshWorktrees();
    } catch (error) {
      set({ 
        error: (error as Error).message, 
        isLoading: false 
      });
      throw error;
    }
  },
  
  // 设置当前仓库
  setCurrentRepo: (repo) => {
    set({ 
      currentRepo: repo,
      worktrees: repo?.worktrees || [],
    });
  },
}));
```

## 七、安全设计

### 7.1 安全原则

1. **最小权限原则**：只请求必要的系统权限
2. **输入验证**：所有用户输入进行校验
3. **路径安全**：防止路径遍历攻击
4. **命令注入防护**：参数化执行外部命令

### 7.2 安全实现

```rust
// src-tauri/src/utils/path.rs

use std::path::{Path, PathBuf};

/// 验证路径是否安全
pub fn validate_path(path: &str) -> Result<PathBuf, anyhow::Error> {
    let path = PathBuf::from(path);
    
    // 规范化路径
    let canonical = path.canonicalize()
        .map_err(|_| anyhow::anyhow!("Invalid path"))?;
    
    // 检查路径是否在用户目录下
    let home = dirs::home_dir()
        .ok_or_else(|| anyhow::anyhow!("Cannot determine home directory"))?;
    
    if !canonical.starts_with(&home) {
        return Err(anyhow::anyhow!("Path must be under home directory"));
    }
    
    Ok(canonical)
}

/// 清理分支名，防止注入
pub fn sanitize_branch_name(name: &str) -> Result<String, anyhow::Error> {
    // Git 分支名规则
    let re = regex::Regex::new(r"^[a-zA-Z0-9_\-/.]+$").unwrap();
    
    if !re.is_match(name) {
        return Err(anyhow::anyhow!("Invalid branch name"));
    }
    
    // 防止路径遍历
    if name.contains("..") {
        return Err(anyhow::anyhow!("Branch name cannot contain '..'"));
    }
    
    Ok(name.to_string())
}
```
